Para que sirve

El uso de esta herramienta permite organizar de mejor manera loos atributos de una pagina

Como invocarla?

se crean elementos con su especificacion de clase o id, luego se especifica el tipo de display que va a tener en el css
para esto se va a utilizar
display: grid;
para crear la base del display lo cual van a ser cajas
grid-template-columns: 25% 25% 20% ;
para definir el size de las cajas o columnas
se recomienda ponerle un fondo y un borde para definir el area
