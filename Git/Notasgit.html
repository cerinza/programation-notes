GitHub


Como subir un codigo a github?


1. se copia el link que aparece en "clone with https"


2. despues de asegurar la posicion de carpeta en git bash con "pwd", se usa

git remote add origin (direccion copiada)

se comprueva con "git remote" y "git remote -v"

3. git push origin (mater o nombre de la rama que se va a enviar), se abrira la ventana de gitHub para poner las credenciales


4." git pull origin master" para traer lo de github y " git pull origin master --allow-unrelated-histories" para corregir el error de documentos no relacionados

5. Despues de haber corregido cualquier error se hace de nuevo "git push origin (mater o nombre de la rama que se va a enviar)"
