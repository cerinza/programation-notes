Funcionalidades

git config = las funcionalidades de git.
git config --list = Para mostrar como esta configurado por defecto el git
git config --list --show-origin= Para ver donde estan las configuraciones guardadas
git config --global user.name "(nombre)" = para configurar el nombre de usuario
git config --global user.email "(email.com)" =para configurar el correo.
cat (nombre del archivo)= para ver el contenido actual
code + (nombre del archivo) =abrir el archivo en editor de codigo
